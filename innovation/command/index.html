    新创指令系统文档¶

    指令命令行界面说明¶

    1. 进入方式：点击右上角按键，可进入指令模式。

    2. 进入指令模式后，上方为文本显示区，下方为文本输入栏，点击右上角按键（或输入Quit指令、按ESC键）可退出指令模式。

    3. 指令输入是大小写不敏感的，但部分特殊场合可能会有大小写敏感的情况。

    4. 进入游戏后，指令模式可在任何游戏模式下进入。但目前的大部分指令都是用于建造模式下的，如果在其它模式下使用这些指令，可能会造成意想不到的后果。

    5. 按ESC键可以退出指令模式，按上箭头 / 下箭头键可在输入框内显示之前输入的指令内容。

    6. 如果发生了指令参数错误或者指令运行错误，将报错，文本显示区将显示报错信息。

    7. 如果指令运行无效但未显示报错信息，可能是因为指令名输入错误。

    指令语法格式说明¶

    1. 一条指令分为指令名 / 选项 / 参数三个部分。 其中，指令名代表所调用对应指令的名称，而选项与参数的设定则分别根据对应指令而定。

    2. 每条指令间以换行符或分号来作为分隔符，单条指令中各项间以空格来作为分隔符。

    3. 在指令说明中，会采用以下符号形式来对指令语法项进行说明：

      符号 类型 说明
      <...> 尖括号内文本 必选的选项或参数（一般内含选项名或参数名）
      [...] 方括号内文本 可选的选项或参数（一般内含选项名或参数名）
      ... 无括号文本 必须输入的文本项（一般为指令名）
      -... 连字符后跟文本 选项名（通常为单个字符）

    4. 在输入指令时，语法中的选项名不需要替换，参数名需要替换为对应参数值。

    5. 一般地，各选项之间的顺序可以更换，各参数之间的顺序不能更换。

      有时，一个可选参数会与一个选项搭配成一组，这样的参数组合就具有了可更换顺序的功能。

    6. 在指令说明中，会对指令参数的数据类型进行说明。一般有以下数据类型：

      类型 说明
      bool 布尔值类型，输入值为 True / False
      byte 整型类型，输入值为 0 到 255 的无符号 8 位整数
      int 整型类型，输入值为 -2147483648 到 2147483647 的带符号 32 位整数
      float 浮点类型，输入值为单精度浮点数
      enum 枚举类型，一般用于定义部件类型、配置选项类型等内容，输入值为特定字符串或整数
      string 字符串类型，表示多个字符的序列
      T 泛型，非特定数据类型

    7. 语法说明示例：

      GetPartPosition [-a] <partType>

      在该部件位置获取指令的语法中， GetPartPosition 是必须输入的文本项，代表指令名。

      [-a] 是可选的选项，其中连字符后跟选项名 a ；使用该选项时，直接输入 -a 即可。

      <partType> 是必选的参数；使用该参数时，需要输入对应参数值。

    指令说明（语法 / 说明 / 参数 / 示例）¶

    一、指令系统设置类¶

    1. Quit

      语法：Quit

      说明：退出指令模式

      适用范围：全局

      选项与参数：无

    2. Clear

      语法：Clear

      说明：清除文本显示区的文本，同时会取消选择部件

      适用范围：全局

      选项与参数：无

    3. SetButtonAlpha

      语法：SetButtonAlpha <alpha>

      说明：设置指令按键的贴图不透明度

      适用范围：全局

      选项与参数：

      • alpha：float类型，指令按键的不透明度值

      示例：SetButtonAlpha 0 将使按键贴图完全透明但点击依然有效，相当于隐藏GUI效果

      备注：指令按键的不透明度默认值为255，相当于完全不透明

    4. SetButtonPosition

      语法：SetButtonPosition <x> <y>

      说明：设置指令按键的位置

      适用范围：全局

      选项与参数：

      • x：float类型，按键横坐标
      • y：float类型，按键纵坐标

      示例：SetButtonPosition 0 -984 如果屏幕分辨率为1920x1080，那么该指令可将按键移至右下角（按键大小为128x96，因此y值为-1080+96=-984）

      备注：指令按键坐标的默认值为(0, 0)；x以向右为正方向，y以向上为正方向

    二、基础语法函数类¶

    1. Define

      语法：Define <identifier> <replacement>

      作用说明：宏定义，支持文本宏替换

      适用范围：全局

      选项与参数：

      • identifier：string类型
      • replacement：string类型

    三、基础设施交互类¶

    1. GetSettingsValue

      语法：GetSettingsValue <option>

      说明：获取自定义配置选项的设定值

      适用范围：全局

      选项与参数：

      • option：enum类型，对应自定义配置选项的名称

      示例：GetSettingsValue NoGravity 将获取无重力配置选项的对应值 True / False

    2. SetSettingsValue

      语法：SetSettingsValue <option> <value>

      说明：设置自定义配置选项的设定值

      适用范围：全局

      选项与参数：

      • option：enum类型，对应自定义配置选项的名称
      • value：T类型，将设置的设定值

      示例：SetSettingsValue NoGravity True 将开启无重力配置选项

      备注：设定值的类型与配置选项相对应，如果类型不匹配将报错；设置后不会自动保存配置，如果想要保存则需使用SaveSettings指令；部分配置选项可能无法运行时更改

    3. GetSettingsScope

      语法：GetSettingsScope <option>

      说明：获取自定义配置选项的作用域

      适用范围：全局

      选项与参数：

      • option：enum类型，对应自定义配置选项的名称

    4. SetSettingsScope

      语法：SetSettingsScope <option> <scope>

      说明：设置自定义配置选项的作用域

      适用范围：全局

      选项与参数：

      • option：enum类型，对应自定义配置选项的名称
      • scope：enum类型，将设置的作用域

    5. SaveSettings

      语法：SaveSettings

      说明：保存自定义配置选项文件

      适用范围：全局

      选项与参数：无

    四、部件属性类¶

    1. GetPartPosition

      语法：GetPartPosition [-a] <partType> [customIndex]

      说明：获取对应类型部件的真实位置坐标

      适用范围：建造模式与运行模式

      选项与参数：

      • -a：all选项，若开启则输出所有对应部件的坐标，若关闭则输出最后放置的部件的坐标
      • partType：enum类型，对应部件类型
      • customIndex：int类型，对应部件材质类型（可选，若不选则默认所有部件材质类型）

      示例：GetPartPosition -a Pig 将获取所有雀斑猪的真实位置坐标

    2. GetPartCoord

      语法：GetPartCoord [-a] <partType> [customIndex]

      说明：获取对应类型的部件的建造位置坐标

      适用范围：建造模式与运行模式

      选项与参数：

      • -a：all选项，若开启则输出所有对应部件的坐标，若关闭则输出最后放置的部件的坐标
      • partType：enum类型，对应部件类型
      • customIndex：int类型，对应部件材质类型（可选，若不选则默认所有部件材质类型）

      示例：GetPartCoord -a Pig 将获取所有雀斑猪的建造位置坐标

    五、部件操作类¶

    1. SelectParts

      语法：SelectParts <x> <y> <width> <height> [partType] [customIndex]

      说明：选择建造格中一部分区域内的部件

      适用范围：建造模式

      选项与参数：

      • x：int类型，矩形选择范围的左下角横坐标
      • y：int类型，矩形选择范围的左下角纵坐标
      • width：int类型，矩形选择范围的宽度
      • height：int类型，矩形选择范围的高度
      • partType：enum类型，所选择的部件类型（可选，若不选则默认所有部件类型）
      • customIndex：int类型，所选择的部件材质类型（可选，若不选则默认所有部件材质类型）

      示例：SelectParts 1 1 10 10 即可选择一个起始点(1,1)、长10格、宽10格的矩形内的所有部件

      备注：如果在已经选择过一次部件后再次选择，原有的选中部件不会被取消选择，选择区域会叠加

    2. DeselectParts

      语法：DeselectParts

      说明：取消选择部件

      适用范围：建造模式

      选项与参数：无

    3. SetParts

      语法：SetParts <x> <y> <width> <height> <partType> [customIndex]

      说明：批量摆放部件

      适用范围：建造模式

      选项与参数：

      • x：int类型，矩形选择范围的左下角横坐标
      • y：int类型，矩形选择范围的左下角纵坐标
      • width：int类型，矩形选择范围的宽度
      • height：int类型，矩形选择范围的高度
      • partType：enum类型，所摆放的部件类型
      • customIndex：int类型，所摆放的部件材质类型（可选，若不选则默认无星材质类型）

      示例：SetParts 1 1 10 10 MetalFrame 0 即可在一个起始点(1,1)、长10格宽10格的矩形内批量摆放无星铁框部件

    4. SetPartsInterval

      语法：SetPartsInterval <x> <y> <width> <height> <deltaX> <deltaY> <partType> [customIndex]

      说明：按一定间距批量摆放部件

      适用范围：建造模式

      选项与参数：

      • x：int类型，矩形选择范围的左下角横坐标
      • y：int类型，矩形选择范围的左下角纵坐标
      • width：int类型，矩形选择范围的宽度
      • height：int类型，矩形选择范围的高度
      • deltaX：int类型，水平摆放间距
      • deltaY：int类型，垂直摆放间距
      • partType：enum类型，所摆放的部件类型
      • customIndex：int类型，所摆放的部件材质类型（可选，若不选则默认无星材质类型）

      示例：SetPartsInterval 1 1 10 10 2 2 MetalFrame 0 即可在一个起始点位于(1,1)、长10格、宽10格的矩形内间距为二（即相邻部件之间隔一格）地批量摆放无星铁框部件

      备注：当deltaX / deltaY参数为1时，与SetParts效果相同；当deltaX / deltaY参数为非正整数时，将报错

    5. MoveParts

      语法：MoveParts <x> <y>

      说明：移动选中的部件到指定位置

      适用范围：建造模式

      选项与参数：

      • x：int类型，水平移动距离
      • y：int类型，垂直移动距离

      示例：MoveParts 10 10 即可将选中部件移动到向右10格、向上10格的位置

      备注：使用该指令后，选中部件并不会被取消选择；此指令使用前需搭配SelectParts指令

    6. CopyParts

      语法：CopyParts <x> <y> [reselect]

      说明：复制选中的部件到指定位置

      适用范围：建造模式

      选项与参数：

      • x：int类型，水平复制距离
      • y：int类型，垂直复制距离
      • reselect：bool类型，是否重新选择，将选择范围替换为复制后的新部件（可选，若不选则默认不替换）

      示例：CopyParts 10 10 即可将选中部件复制到向右10格、向上10格的位置

      备注：使用该指令后，选中部件并不会被取消选择（除非进行了重新选择）；此指令使用前需搭配SelectParts指令

    7. RotateParts

      语法：RotateParts <count>

      说明：翻转或旋转选中的部件

      适用范围：建造模式

      选项与参数：

      • count：int类型，翻转 / 旋转的次数

      备注：此指令使用前需搭配SelectParts指令

    8. ReplaceParts

      语法：ReplaceParts <partType> [customIndex] [reselect]

      说明：将选中的部件替换为指定类型的新部件

      适用范围：建造模式

      选项与参数：

      • partType：enum类型，所替换成的部件类型
      • customIndex：int类型，所替换成的部件材质类型（可选，若不选则默认无星材质）
      • reselect：bool类型，是否重新选择而将替换后的新部件纳入选择范围（可选，若不选则默认不替换）

      示例：ReplaceParts MetalFrame 0 即可将选中部件替换为无星铁框

      ​ ReplaceParts MetalFrame 0 True 即可将选中部件替换为无星铁框，同时将替换上去的无星铁框纳入部件选中区域

      备注：此指令使用前需搭配SelectParts指令；使用该命令后将清除部件选中区域；如果开启了重新选择选项，那么清除后会再将新部件选择

    9. RemoveParts

      语法：RemoveParts

      说明：移除选中部件

      适用范围：建造模式

      选项与参数：无

      备注：此指令使用前需搭配SelectParts指令；使用该命令后将清除部件选中区域
